Installation Guide

Follow these steps to install lexOS on your machine:

  1. 0 Make a user named lex (on linux) and make it root (for more information why you need to do this read the github)
  3. 1 Download lexium VM and lexOS-image
  4. 2 Run lexium with normal settings (80% of your cpu will be used)
  5. 3 For running lexOS run the RUNER executable
  6. 4 The user will be lex and the password should be your user password and if that doesn't work try gulb12, i32, sun12, u32 or 1234
